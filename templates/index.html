About This Project

A Machine Learning web application that predicts house prices using the Random Forest Regression algorithm.

Project Overview

The House Price Predictor is developed using Python, Flask, HTML, CSS, Bootstrap, and Machine Learning. The application predicts the estimated selling price of a house based on important property features provided by the user.

The prediction model is trained using the Kaggle House Prices dataset, which contains 1,460 real-world house records with multiple housing attributes.

Random Forest Regression was selected because it provides better accuracy than Linear Regression and Decision Tree Regression for this dataset. The trained model achieved an accuracy of approximately 89%.

How It Works

The prediction process is simple and efficient.

Step 1

Enter the house details such as quality, area, garage capacity, bathrooms, basement area, and year built.

Step 2

The Random Forest Regression model processes the input values.

Step 3

The trained machine learning model predicts the estimated house price.

Step 4

The estimated price is displayed instantly along with model details.
